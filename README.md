# Terraform

## Quick install on WSL2
```
 sudo apt  install unzip -y
 curl -O https://releases.hashicorp.com/terraform/0.14.6/terraform_0.14.6_linux_amd64.zip
 unzip terrafor*
 sudo mv terraform /usr/local/bin
 rm -f terra*linux*zip
```
